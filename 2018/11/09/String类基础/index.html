一、String字符串的创建方式

1、直接赋值：

​ String str = "字符串";

2、也可以用构造方法：

​ String str = new String()

二、字符串拼接

字符串可以用“+”号拼接：

​ String str = "字符串1" + "字符串2"

​ 但要注意：这是创建了一个新的字符串，并不是真正意义上的相加，

原来的“字符串1”的存储空间还在，详情查看关于内存分析的博客。

所以一般使用StringBuffer或StringBuilder，关于这两个类也在其它博客。

8、substring(int beginIndex) 和substring(int beginIndex, int endIndex)：

这个方法返回一个新字符串，它是此字符串的一个子字符串，这个的索引取值范围是左闭有开区间,

如substring(0,3)，所取到的值是0、1、2不包括3。

substring(2),所取到的值是索引为2的字符到该字符串最后一个字符

9、toCharArray()

将此字符串转换为一个新的字符数组。

10、toLowerCase()

使用默认语言环境的规则将此 String 中的所有字符都转换为小写。

11、toUpperCase()

使用默认语言环境的规则将此 String 中的所有字符都转换为大写。

12、trim()
